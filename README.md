# VrbDespesas
 Software de controle e gerenciamento de despesas.
 
## Tecnologias utilizadas:
* <a href="https://www.lazarus-ide.org/">Lazarus FPC (Versão 3.4).</a>

## Arquitetura:
* Padrão MVC
